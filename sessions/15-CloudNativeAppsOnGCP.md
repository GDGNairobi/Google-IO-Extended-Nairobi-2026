# Session 14 - Deploying Cloud-Native Apps on GCP with Antigravity CLI

- **Speakers**: [Alex Nyambura](https://www.linkedin.com/in/lxmwaniky/)

## About the session
Most developers know how to write code, but few know how to ship it reliably. Between setting up Cloud Run, configuring Pub/Sub, writing Terraform, and managing IAM, the DevOps layer is where most developers and small teams get stuck.

Antigravity CLI, announced at Google I/O 2026, changes that. In this session, I will go beyond the documentation to share my hands-on experience using the tool to design, provision, and deploy a production-grade serverless application on GCP from a single prompt.

During this live demonstration, I will show AGY in action as it plans architecture, generates infrastructure code, deploys to Cloud Run, and verifies the pipeline end-to-end. We will have an honest discussion about its current capabilities, where human judgment remains critical, and the impact this has on solo developers and small engineering teams across Africa who lack dedicated DevOps resources.

Attendees will leave with a clear mental model of how to leverage Antigravity CLI and a codelab they can run immediately to start simplifying their own deployment workflows.
