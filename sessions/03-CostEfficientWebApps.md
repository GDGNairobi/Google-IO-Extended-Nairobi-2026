# Session 3 - Smarter, Cheaper, Faster: Building Cost-Efficient Web Apps with Firebase AI Logic & Chrome Hybrid Inference

- **Speakers**: [Auwal MS](https://www.linkedin.com/in/auwalms/)

## About the session
Want to deploy powerful generative AI features on the web without breaking your cloud budget or failing on spotty networks? In this session, we will unpack the game-changing Google I/O 2026 updates to Firebase AI Logic. Attendees will discover how Web Hybrid Inference lets them execute lightning-fast, zero-cost AI prompts locally in Chrome using on-device models, with a seamless cloud fallback to Gemini 3.x models when needed. We'll also cover how to keep client applications highly secure using Firebase's new Template-Only Mode to completely neutralize prompt injection attacks.
