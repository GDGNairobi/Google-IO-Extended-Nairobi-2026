# Session 8 - Building a World Cup Analytics Agent using the Managed Agents API and Agent Platform

- **Speakers**: [Gabriel Agbobli](https://www.linkedin.com/in/gabriel-agbobli-3966801a1/)

## About the session
Using the Managed Agents API on the Gemini Enterprise Agent Platform, this hands-on session will demonstrate how to build and deploy an intelligent World Cup analytics assistant capable of reasoning over historical and real-time football data. Participants will learn how to create and manage enterprise-grade agents through REST APIs, configure custom tools and skills, provision managed execution environments, and build persistent multi-turn conversational experiences.

Throughout the session, attendees will deploy a World Cup analysis agent that combines Google Search, file system tools, Google Cloud Storage-hosted skills, and external data sources to answer complex football-related questions. The workshop will also showcase agent lifecycle management, environment persistence, interaction history, and reconnection to active sessions, enabling developers to build stateful AI applications that maintain context across conversations.

By the end of the session, participants will understand how to use the Managed Agents API to orchestrate tools, manage agent configurations, deploy custom skills, and create sophisticated conversational agents capable of performing data analysis, generating insights, and interacting with users through natural multi-turn dialogue. The concepts covered extend beyond sports analytics and provide a foundation for building enterprise AI agents across customer support, business intelligence, operations, and knowledge management use cases.
