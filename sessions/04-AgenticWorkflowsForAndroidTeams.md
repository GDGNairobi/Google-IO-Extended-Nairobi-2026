# Session 4 - Beyond the Prompt: Building Strict, High-Quality Agentic Workflows for Android Teams

- **Speakers**: [Jacquiline Gitau](https://www.linkedin.com/in/jacquiline-gitau/)

## About the session
The biggest fear for any Android developer in 2026 isn't AI—it’s "AI slop": technically functional code that violates architecture, ignores team conventions, and creates technical debt. With Google I/O 2026, we now have the tools to move from unmonitored "vibe coding" to a rigorous, team-wide agentic workflow.

In this talk, I’ll demonstrate how to set up Android Skills to enforce your team’s specific architecture across all developer machines. We’ll dive into the new Smart Code Review tool in Android Studio Quail, which uses specialised sub-agents to catch "slop" before it ever reaches a human reviewer. I will show how to use the Model Context Protocol (MCP) to connect your agents to your team's internal documentation and ensure the code generated is high-quality and context-aware.
