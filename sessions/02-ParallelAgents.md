# Session 2 - Orchestrating Parallel Agents with Antigravity 2.0

- **Speakers**: [Shadrack Inusah](https://www.linkedin.com/in/shadrack-inusah/)

## About the session
Google I/O 2026 introduced a major evolution in AI-assisted development with the launch of Antigravity 2.0, a standalone desktop application designed for autonomous software engineering. In this session, I will explore how developers can move beyond code completion and leverage Gemini 3.5 Flash, Dynamic Subagents, Scheduled Tasks, and JSON hooks to build intelligent workflows that run independently in the background. Through a live demo, we will configure an autonomous data-processing pipeline that schedules tasks, launches specialized agents in parallel, and generates reviewable outputs without interrupting the main workspace. Attendees will gain practical experience with multi-agent orchestration, workflow automation, and secure local integrations and learn how these capabilities can help lean engineering teams scale productivity and streamline operations.
