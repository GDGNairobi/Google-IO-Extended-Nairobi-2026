# Session 7 - Robust Queries, Semantic Search and Geospatial Retrieval with Firestore Pipelines

- **Speakers**: [Hassan Bahati](https://www.linkedin.com/in/hassanbahatim/)

## About the session
Firestore is evolving beyond a traditional NoSQL database into a powerful foundation for intelligent data retrieval. With Firestore Pipelines, developers can build search experiences that go beyond exact keyword matching, enabling applications to understand meaning, context, and user intent while leveraging the simplicity and scalability of Firebase.

In this session, we'll explore how Firestore Pipelines unlock advanced retrieval capabilities, including semantic search, full-text search, pipeline operations, and richer query patterns that power modern AI-driven applications. Attendees will learn how to transform Firestore data into searchable knowledge, generate and manage embeddings, and implement retrieval systems that surface relevant results based on both meaning and traditional search criteria.

Through practical demonstrations, we'll examine how to combine semantic retrieval with structured queries, geospatial filtering, and ranking strategies to build smarter discovery experiences. We'll also discuss how these capabilities fit into modern AI architectures, including Retrieval-Augmented Generation (RAG), recommendation systems, content discovery platforms, and location-aware applications.

By the end of the session, attendees will understand how to leverage Firestore Pipelines as part of Firebase's growing intelligent data platform, enabling applications that can search, retrieve, and reason over data more effectively while remaining fully integrated within the Firebase ecosystem.
