# Session 14 - Stay Guided, Reduce the Slop: Bridging the knowledge gap in modern Web Development

- **Speakers**: [Rama Ochieng](https://www.linkedin.com/in/ramaochieng/)

## About the session
We are rapidly entering the "agentic era" of modern frontend development, yet AI assistants drag us backward with legacy "slop" due to knowledge cutoffs from their training data. While the Chrome team shipped 50+ new web features recently, AI still recommends bloated JavaScript for UI components needing zero JS. In our African ecosystem, where data efficiency is critical, we cannot afford this bloat.

This session explores Modern Web Guidance, an expert-vetted tool announced at Google I/O 2026. It intercepts AI prompts via local semantic searches against 100+ specific guides. I will share my direct experience using it to eliminate AI hallucinations.

Through a live coding demo, I will demonstrate how this tool forces agents to build lightweight, cross-browser experiences by comparing "guided" vs "unguided" outputs. Attendees will learn to install it via a single CLI command and use an AGENTS.md file for explicit browser fallbacks, leaving ready to build hyper-efficient web apps.
