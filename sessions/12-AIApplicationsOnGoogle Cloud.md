# Session 11 - Build, Deploy, Scale: Creating AI Applications with Google Cloud

- **Speakers**: [RHODAH MULERA](https://www.linkedin.com/in/rhodah-mulera/)

## About the session
AI is changing how developers build applications, and Google Cloud provides the tools to bring intelligent ideas into production faster than ever. In this session, we’ll explore how to build and deploy AI-powered applications using Google AI Studio, Cloud Run, Firebase, and Gemini APIs.

Through a live hands-on demo, attendees will learn how to integrate generative AI into a real-world web application, deploy scalable backend services on Google Cloud, and create modern cloud-native architectures suitable for startups, student projects, and production systems.

Whether you are a beginner exploring cloud technologies or an intermediate developer looking to integrate AI into your applications, this session will provide practical workflows, live coding demonstrations, and deployment strategies using Google’s latest AI and cloud ecosystem.
